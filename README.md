# CSGO AI自动瞄准


## 使用步骤：

1. 从百度网盘下载：

    链接: https://pan.baidu.com/s/1R8oICRDjAVY82vVLsl7W2w 
    
    提取码: csgo

2. **解压之后双击运行 `AI瞄准启动器`。**     注意：为了避免二次售卖,所以增加卡密校验，但是卡密是免费获取的，无限额度，每日清空。微信小程序搜索“西瓜去水印文案配音助手”,点击获取游戏卡密条目即可

3. 首次启动时，AI模型将被导出（可能需要几分钟）。

4. 使用 Home 键打开/关闭叠加层。所有设置均可在叠加层中找到。



## 按键控制

1. 鼠标右键：**瞄准已检测到的目标
2. F2：**退出
3. F3：**暂停瞄准
4. F4：**重新加载配置
5. Home键：**打开/关闭叠加层和设置
